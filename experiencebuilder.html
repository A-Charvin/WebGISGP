ArcGIS Experience Builder

Engaging & User-Friendly Tree Data Visualization

ArcGIS Experience Builder provides a customizable, user-friendly, and interactive platform for presenting tree data in an engaging manner. Its ability to integrate various datasets, create rich visualizations, and incorporate interactive components makes it an ideal solution for addressing the reality that tree data is not presented in an interesting, attractive and functional manner.

ArcGIS Experience Builder: Peterborough's Urban Forest
2D & 3D Visualization Options

Ability to create a variety of interactive visualizations, including 2D & 3D maps: With Experience Builder, interactive 2D and 3D maps provide a more immersive and engaging experience for users. User's can explore, navigate, and interact with the data in ways that static visualizations cannot offer, encouraging them to spend more time on the Web Application.

Improved understanding of spatial relationships: 3D visualizations offer an accurate portrayal of real-world landscapes and structures, greatly benefiting tree data representation online. Meanwhile, 2D visualizations effectively contextualize the entire study area.

Dashboard & Timeline Widget

Dashboard: Dashboards are optimal for visualizing tree data as they offer a real-time, consolidated view of key metrics, allowing users to easily explore, analyze, and make informed decisions regarding urban forestry initiatives. This user-friendly and interactive approach promotes effective management and a better understanding of the city's tree resources.

Timeline widget: Enables the visualization of temporal data from web maps, feature layers, and map service layers, showcasing how tree data changes over time. This widget allows users to animate tree data using play and pause buttons or manually view changes with a slider, and zoom in and out on the timeline, enhancing their understanding of tree data trends and patterns.

Data Editing & View's Navigation

Data Editing: Experience Builder supports editing features within feature layers directly from the web application. This capability enables users to add, modify, or delete features, making it possible to collect and update data in real-time, fostering collaboration and accurate data management.

View's Navigation: With Views Navigation, the user can be guided through different visualizations and data sets, providing them with a streamlined experience. This simplification helps users focus on specific aspects of the tree data without being overwhelmed by information. The ability to switch between different perspectives of tree data seamlessly, and by providing contextual navigation, the users gain a comprehensive understanding of the tree data.
